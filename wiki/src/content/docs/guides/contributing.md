LuminusOS is built by volunteers, in the open, on top of Fedora. Every issue,
suggestion and pull request shapes the project.

## Where everything lives

| Repository | What it is |
| --- | --- |
| [luminusOS/images](https://github.com/luminusOS/images) | The OS images (Fedora bootc), build system and CI |
| [luminusOS/aurora-shell](https://github.com/luminusOS/aurora-shell) | The Aurora Shell GNOME extension |
| [luminusOS/website](https://github.com/luminusOS/website) | luminusos.org |
| [luminusOS/wiki](https://github.com/luminusOS/wiki) | This wiki |
| [luminusOS/blog](https://github.com/luminusOS/blog) | blog.luminusos.org |

## Ways to help

- **Test and report.** Run Workstation, file issues for anything weird. A good
  bug report is a contribution.
- **Write docs.** This wiki is young. Every page has an "Edit page" link that
  takes you straight to GitHub.
- **Translate.** The website and this wiki speak English and Portuguese so
  far. More languages are welcome.
- **Code.** The images repo and Aurora Shell both have good first issues.
- **Support the project.** [Donations](https://opencollective.com/luminusos)
  help us ship the next edition sooner.

## Future editions

Play (handhelds), Mobile (phones), Tablet and Cast (TVs) are reserved in the
build system and waiting for contributors. If one of those screens matters to
you, come build it with us.
