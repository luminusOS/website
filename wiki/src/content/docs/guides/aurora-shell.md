[Aurora Shell](https://aurora.luminusos.org) is a modular GNOME Shell
extension made by the LuminusOS project. It adds the quality-of-life features
missing in vanilla GNOME and ships preinstalled on Workstation.

## The modules

Every module is independent. Enable exactly what you want in the extension
preferences (Extensions app → Aurora Shell → Settings).

| Module | What it does |
| --- | --- |
| Dock | Smart per-monitor dock with intellihide and edge reveal |
| Volume Mixer | Per-app volume sliders in Quick Settings |
| Tray Icons | Real system tray with SNI icons and background apps |
| Weather Clock | GNOME Weather next to the panel clock |
| Meeting Clock | Next calendar event beside the clock, with reminders |
| No Overview | Boot straight to the desktop |
| PiP On Top | Keeps Picture-in-Picture windows above everything |
| Theme Changer | Consistent light/dark color-scheme behavior |
| DND on Screen Share | Auto Do-Not-Disturb while sharing or recording |
| XWayland Indicator | Marks XWayland apps in the window switcher |
| Icon Weave | Fixes missing app icons automatically |

## On other distros

Aurora Shell works on any distro with GNOME. Install it from
[extensions.gnome.org](https://extensions.gnome.org/extension/9389/aurora-shell/).

## Privacy

Everything stays on your machine. No module ever sends data to third parties.
