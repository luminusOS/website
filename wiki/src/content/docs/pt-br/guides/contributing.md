O LuminusOS é feito por voluntários, abertamente, em cima do Fedora. Cada
issue, sugestão e pull request molda o projeto.

## Onde tudo mora

| Repositório | O que é |
| --- | --- |
| [luminusOS/images](https://github.com/luminusOS/images) | As imagens do SO (Fedora bootc), sistema de build e CI |
| [luminusOS/aurora-shell](https://github.com/luminusOS/aurora-shell) | A extensão Aurora Shell para o GNOME |
| [luminusOS/website](https://github.com/luminusOS/website) | luminusos.org |
| [luminusOS/wiki](https://github.com/luminusOS/wiki) | Esta wiki |
| [luminusOS/blog](https://github.com/luminusOS/blog) | blog.luminusos.org |

## Jeitos de ajudar

- **Teste e relate.** Use a Workstation e abra issues para qualquer coisa
  estranha. Um bom relato de bug é uma contribuição.
- **Escreva documentação.** Esta wiki é nova. Toda página tem um link
  "Editar página" que leva direto ao GitHub.
- **Traduza.** O site e esta wiki falam inglês e português por enquanto.
  Mais línguas são bem-vindas.
- **Programe.** O repo das imagens e o Aurora Shell têm boas issues para
  começar.
- **Apoie o projeto.** [Doações](https://opencollective.com/luminusos)
  ajudam a próxima edição a chegar mais cedo.

## Edições futuras

Play (portáteis), Mobile (celulares), Tablet e Cast (TVs) estão reservadas no
sistema de build esperando contribuidores. Se uma dessas telas importa para
você, vem construir com a gente.
