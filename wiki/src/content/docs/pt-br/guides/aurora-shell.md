O [Aurora Shell](https://aurora.luminusos.org) é uma extensão modular do
GNOME Shell feita pelo projeto LuminusOS. Ela adiciona os recursos de
qualidade de vida que faltam no GNOME puro e vem pré-instalada na Workstation.

## Os módulos

Cada módulo é independente. Ative exatamente o que quiser nas preferências da
extensão (app Extensões → Aurora Shell → Configurações).

| Módulo | O que faz |
| --- | --- |
| Dock | Dock inteligente por monitor com intellihide e revelação na borda |
| Volume Mixer | Volume por aplicativo nas Configurações Rápidas |
| Tray Icons | Bandeja de sistema de verdade, com ícones SNI e apps em segundo plano |
| Weather Clock | GNOME Clima ao lado do relógio do painel |
| Meeting Clock | Próximo evento da agenda ao lado do relógio, com lembretes |
| No Overview | Inicia direto na área de trabalho |
| PiP On Top | Mantém janelas Picture-in-Picture acima de tudo |
| Theme Changer | Comportamento consistente de tema claro/escuro |
| DND on Screen Share | Não Perturbe automático ao compartilhar ou gravar a tela |
| XWayland Indicator | Marca apps XWayland no alternador de janelas |
| Icon Weave | Conserta ícones de apps ausentes automaticamente |

## Em outras distros

O Aurora Shell funciona em qualquer distro com GNOME. Instale pelo
[extensions.gnome.org](https://extensions.gnome.org/extension/9389/aurora-shell/).

## Privacidade

Tudo fica na sua máquina. Nenhum módulo envia dados para terceiros.
